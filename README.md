# Math
Allows to compute:
- the least common multiple
- the greatest common divisor

## Usage
```php
<?php
use lukaszmakuch\Math;

//the least common multiple
echo Math::lcm(3, 4); //12

//the greatest common divisor
echo Math::gcd(8, 12); //4
```

## Getting the library
```
$ composer require lukaszmakuch/math
```
